# CV Maker

**By Mohamed Sabiullah**

![License](https://img.shields.io/badge/license-MIT-blue.svg)

CV Maker is a free, powerful, and privacy-first online resume generator. Write your curriculum vitae (CV) and resumes seamlessly with Markdown!

## Features

- **Simple and Powerful:** Build and manage multiple resumes with ease.
- **Markdown Support:** Write entirely in Markdown with a live preview.
- **Export to PDF:** Perfectly formatted exports to PDF.
- **Customizable:** Change margins, colors, and configure it completely to your needs.
- **Privacy First:** All your data remains perfectly safe inside your own browser window. No login, no users tracking, and no ads.
- **Multi-language Support:** Choose the language that works best for you (English, Spanish, Chinese).

## Tech Stack

This project is built using modern front-end technologies:

- **Framework:** Nuxt 3 (Vue.js 3 ecosystem)
- **Styling:** UnoCSS & Tailwind CSS (with Shadcn-Vue components)
- **State Management:** Pinia
- **Editor:** Monaco Editor
- **Markdown engine:** \`markdown-it\` with custom plugins for LaTeX (\`KaTeX\`)
- **Local Storage:** LocalForage

## Getting Started

To run CV Maker locally on your machine, follow these simple steps:

1. **Clone the repository:**
   \`\`\`bash
   git clone https://github.com/sabireem/cv-maker.git
   cd cv-maker
   \`\`\`

2. **Install Dependencies:**
   Make sure you have [pnpm](https://pnpm.io/) installed.
   \`\`\`bash
   pnpm install
   \`\`\`

3. **Start the Development Server:**
   \`\`\`bash
   pnpm run dev
   \`\`\`

4. **Build for Production:**
   \`\`\`bash
   pnpm run build
   \`\`\`
